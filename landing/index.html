How to install the Android APK
  1. Tap the download button — your browser saves the .apk file.
  2. Open the downloaded file. Android will ask permission to install from this source — tap "Allow" or "Settings → enable for Chrome".
  3. Google Play Protect may warn that "the developer isn't verified". That's normal for direct APK installs — tap "Install anyway".
  4. Open Mood8 and sign in. Done.

How to install on Wear OS (technical)

Wear OS doesn't have a sideload-from-watch flow — installing a Wear APK is a developer process. Intended for technical testers, not casual users.

  1. On the watch: Settings → System → About → Versions — tap the build number 7 times to enable Developer options.
  2. Back in Settings, open Developer options and turn on ADB debugging and Debug over Wi-Fi. The watch will show an IP address.
  3. On your computer (with adb installed): adb connect <watch-ip>:5555 Accept the prompt on the watch.
  4. Download mood8-wear.apk and install it: adb install mood8-wear.apk
  5. The app appears in the watch's launcher. Sign in with the same account as the phone — your check-ins sync.

If you don't recognise half the words above, stick with the phone app — the Wear companion adds quick check-ins but doesn't unlock anything you can't already do on phone.
