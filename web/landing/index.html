Frequently asked

Is my data private?

Yes. All your data is stored locally in your browser. We only send anonymized requests to OpenAI when you use AI features (you can disable this). We don't sell data. Ever.

Does it work offline?

Most features work offline since data is stored locally. AI Coach requires internet (OpenAI API). You can check in, track habits, and view progress completely offline.

Can I export my data?

Yes! Settings → Data & Privacy → Export. Choose JSON (full backup) or CSV (spreadsheet-friendly). Your data, your control.

Does it really use AI?

Yes. We use ChatGPT (gpt-4o-mini) for reflections and chat. Statistical analysis for insights runs locally — no AI needed for pattern detection.

Is it open source?

Coming soon. We're considering open-sourcing the client. Backend stays closed for API key protection.

What about Apple Watch?

Wear OS (Samsung, Pixel) is supported now. Apple Watch is in development — requires Apple Developer account.

Why "Mood8"?

8 is infinity sideways — endless growth. Also represents the 8 dimensions we track: mood, energy, focus, routine, habits, identity, time, and self.
